1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
   ans: getElementById select the elements by id name, getElementsByClassName select the all elements by same class name, queryselector selects the first matching element only, querySelectorAll selects the all matching elements... 
2. How do you create and insert a new element into the DOM?
    ans: to create a new element into the dom is: document.createElement("p"), and insert the element like container.appendChild("div")
3. What is Event Bubbling? And how does it work?
   Event bubbling is when an event on an element automatically flows up and triggers on its parent elements.
4. What is Event Delegation in JavaScript? Why is it useful?
   Event delegation lets a parent handle events from its children efficiently, especially when elements are dynamic.
5. What is the difference between preventDefault() and stopPropagation() methods?
   preventDefault() stops the browser’s default action, while stopPropagation() stops the event from bubbling or capturing through the DOM
